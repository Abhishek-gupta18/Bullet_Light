# Bullet Light ✨

A beautiful and minimalist bullet journal website built with HTML and CSS. This project demonstrates fundamental web design principles through a clean, interactive, and user-friendly interface.

## 📖 About

**Bullet Light** is a first-semester project that showcases the power of HTML and CSS in creating elegant web applications. The project combines structured markup with modern CSS styling to deliver a smooth user experience for digital bullet journaling.

## 🎯 Features

- ✍️ **Clean Interface** - Minimalist design focused on usability
- 🎨 **Beautiful Styling** - Modern CSS with smooth transitions and animations
- 📱 **Responsive Design** - Works seamlessly across different screen sizes
- 💾 **Organized Layout** - Well-structured HTML for easy navigation
- 🌟 **Interactive Elements** - Enhanced user experience with CSS effects

## 🛠️ Technologies Used

- **HTML5** (85.2%) - Semantic markup and structure
- **CSS3** (14.8%) - Styling, layout, and visual effects

## 🚀 How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Abhishek-gupta18/Bullet_Light.git
   cd Bullet_Light
   ```

2. **Open in your browser:**
   - Simply open `index.html` in your web browser
   - Or use a live server extension in your code editor

3. **Start journaling:**
   - Navigate through the interface
   - Add and organize your bullet points
   - Enjoy the clean design!
